Authentication

API Key

For server-to-server. Create at /api/v1/api-keys with scoped permissions. Pass via X-API-Key header.

JWT Bearer

For browser/mobile. Login at /api/v1/auth/login, pass as Authorization: Bearer. 15-min TTL, refresh via /api/v1/auth/refresh.

Stripe Webhooks

Billing webhooks verified via Stripe-Signature header using your webhook signing secret.
